# Appointment Scheduling System

##  Problem

Manual appointment scheduling leads to double booking and inefficiency.

##  Solution

A Java-based console system to manage doctors, patients, and appointments.

##  Features

* Add Doctors & Patients
* Book Appointments
* Prevent Double Booking
* View Schedules
* Cancel Appointments

##  Technologies

* Java
* OOP
* ArrayList
* LocalDateTime

##  Run

Compile:
javac -d . main/Main.java models/*.java services/*.java utils/*.java

Run:
java main.Main
